# SnapBook- Android Application
SnapBook is an Android app that beautifully blends vintage photography with digital scrapbooking, letting users capture retro-style photos and creatively preserve their memories. It features real-time vintage filters, an interactive scrapbook editor, and seamless offline storage. A perfect fusion of art, nostalgia, and technology.

---

# Key Features:

1. Vintage Photo Booth: Real-time camera filters (Sepia, Polaroid, Kodachrome, B&W, Cyanotype, Vintage).

2. Photo Strip Generator: Automatic 4-photo strips with borders, timestamps, and save options.

3. Scrapbook Journal: Interactive drag-and-drop canvas to design custom pages using photos, text, and doodles.

4. Cross-Feature Integration: Use your vintage photos directly in scrapbook pages.

5. Offline Storage: Works completely offline using SQLite and local file system.

6. Smart Design: Clean, card-based UI with a warm, vintage color palette for intuitive navigation.

---

## Built With:

- Android Studio (Java)
- CameraX API for camera control
- SQLite for local storage
- Glide for image loading
- Firebase (authentication integration)
